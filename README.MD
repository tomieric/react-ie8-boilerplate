# react-ie8-boilerplate

**`配置5小时开发2分钟`** 系列

- [x] webpack
- [x] webpack-dev-server
- [x] react
- [x] less

## USEAGE

```
git clone http://github.com/tomieric/react-ie8-boilerplate.git myproject && cd myproject
```

## start

```
npm start 
```

## build

```
npm run build
```
